# Internship Portal

An internship management portal built with Next.js, TypeScript, Prisma, PostgreSQL, and NextAuth.

## About

This project was originally developed to simplify the internship submission and verification process for a college department. During development, the institution adopted its own ERP-based solution, making this project no longer necessary for its intended use.

Rather than continuing to build a system without a real deployment path, I decided to conclude the project here.

Although it was never deployed, this project became a significant milestone in my journey as a software engineer. It helped me gain hands-on experience with backend development, authentication, database design, API development, validation, file uploads, and building a production-style application from scratch.

## Tech Stack

- Next.js
- TypeScript
- Prisma
- PostgreSQL
- NextAuth
- Zod
- Tailwind CSS
